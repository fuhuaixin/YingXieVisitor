违反法律法规和相关政策 #

违反法律法规和相关政策的行为及内容，包含但不限于以下场景：

  1. 违反宪法确定的基本原则，煽动抗拒或者破坏宪法、法律、行政法规实施；

  2. 危害国家统一、主权和领土完整，泄露国家秘密，危害国家安全，损害国家尊严、荣誉和利益，宣扬恐怖主义、极端主义；

  3. 诋毁民族优秀文化传统，煽动民族仇恨、民族歧视，侵害民族风俗习惯、歪曲民族历史和民族历史人物，损害革命领袖、英雄烈士形象，伤害民族感情，破坏民族团结；

  4. 煽动破坏国家宗教政策，宣扬宗教狂热，危害宗教和睦，伤害信教公民宗教感情，破坏信教公民和不信教公民团结，宣扬邪教、封建迷信；

  5. 危害社会公德，扰乱社会秩序，破坏社会稳定，宣扬淫秽色情、赌博、吸毒，渲染暴力、恐怖，教唆犯罪或者传授犯罪方法；

  6. 教唆、教授、组织他人进行违法乱纪活动，以非法民间组织名义活动的，宣传违法违规物品；

  7. 侵害未成年人合法权益或者损害未成年人身心健康；

  8. 侵犯他人隐私，侮辱或者诽谤他人，侵害他人合法权益；

  9. 侵害他人名誉权、肖像权、知识产权、商业秘密等合法权利；

  10. 含有法律、行政法规、规章、条例以及任何具有法律效力之规范所限制或禁止的其它内容。

抄袭侵权 #

包含但不限于以下场景：

  1. 未经授权使用他人享有著作权的内容，包括但不限于以洗稿、复制拼凑等方式重新整合，在表达上与在先作品存在实质性的相同或近似的内容；

  2. 违规使用「原创」功能的：指在平台申请声明「原创」，但内容被证实不符合原创标准要求的。

无资质发布专业领域内容 #

  1. 未取得互联网新闻信息服务许可，不得发布新闻信息，新闻信息指：时政类新闻信息，包括有关政治、经济、军事、外交等社会公共事务的报道、评论，以及有关社会突发事件的报道、评论；

  2. 未申请健康类专业资质认证的个人和机构，不得发布疾病治疗、用药指导等健康领域专业性强的内容；

  3. 未申请财经类专业资质认证的个人和机构，不得发布股票买卖指导、投资指南等财经领域专业性强的内容；

  4. 我国法律法规要求必须取得特定资质方可从事特定行为的其他情形，依该等法律法规要求处理。

*如希望通过本平台发布上述内容，应向平台申请资质认证，按照领域类型提交相应资质证明，并在资质许可的范围内发布信息。

风险医疗 #

行为或内容涉及专业医疗信息，但真实性不可考证，对人体健康存在潜在风险。包含但不限于以下场景：

  1. 推广非正规的医疗场所或医疗服务，包括未知安全性的药品、保健品、医美项目、医疗活动等；

  2. 宣传违法违规或背离科学常识的医疗行为，如代孕、胎儿性别鉴定、非法器官移植等；

  3. 以非专业身份，发布涉及中药、针灸、艾灸、刮痧、按穴位、拔罐等医疗手段，指导疾病治疗的内容，但真实性安全性无法保证的；

  4. 不符合专业科学常识，对身体状况作出预警恐吓，或用夸张的话术进行解读、夸大治疗效果，如出现「疗效百分百」等内容的。

风险投资 #

以营利、获益为目的，推广可能会给他人带来严重经济损失的理财产品、投资项目等。包含但不限于以下场景：

  1. 荐股荐彩：推荐购买指定的股票、博彩（包含彩票、六合彩、赌球、跑马）等高风险投资产品，吸引他人跟进购买；

  2. 诱导投资：诱导他人参与投资的各类推广话术，包括但不限于P2P、区块链、各种网赚等。

发布谣言或不实内容 #

包含但不限于以下场景：

  1. 明显谣言或虚假：与事实相悖或违背科学常理，凭空杜撰、拼凑、编造或曲解事实，制造耸人听闻的谣言，影响较为恶劣；

  2. 疑似谣言：无可信来源，断章取义或以讹传讹，存在诋毁、人身攻击等负面影响的；

  3. 信息有误 ： 发布内容与事实不一致、关键信息不准确，或歪曲事实等行为。

违规推广 #

包含但不限于以下场景：

  1. 违禁商品推广：法律规定不准私自制造、买卖、使用、持有、储存、运输、进出口的物品及其他平台禁售商品，如枪支弹药、违禁药品、走私商品等；

  2. 有风险商品推广：对人身健康、财产等存在风险的商品，或过分夸大效果、违反基本常识的虚假宣传；

  3. 未知风险商品推广：除上述两种情况外，其他无法明确判断商品用途及功效，但可能存在未知风险的。

流量作弊 #

滥用产品功能进行流量作弊，批量发布重复、无意义的低质内容，恶意刷粉刷赞，影响用户体验、破坏平台规则且损害他人权益的行为。包含但不限于以下场景：

  1. 违规批量发文：

    1. 批量发布重复内容：一段时间内，用非正常手段连续发布大量重复内容，或高度相似内容；

    2. 批量发无意义/低质内容：连续或大量发布无意义元素，包括但不限于无意义图片、符号、表情、乱码、无序文字等；

  2. 违规刷粉刷赞：通过非正常手段进行作弊，获取虚假粉丝、虚假点赞数据的行为。

平台不鼓励 #

引人不适或反感 #

包含但不限于以下场景：

  1. 血腥暴力：对真实的人或动物实施暴力，煽动他人施暴；内容中有大量血腥画面，未经过马赛克遮挡，引起用户不适，如尸体、内脏器官等；

  2. 密集恐惧：密集排列、分布的事物，造成感官不适；

  3. 猎奇：以猎奇为目的，描述或展现异常事物，引起不适或有不良诱导；

  4. 恶心：引起强烈刺激的事物，如排泄物，吃活物、体验恶心食物等；

  5. 恐怖：内容惊悚的事件，或让人产生恐惧心理的生物，具有令人不寒而栗的特征，如人尸、蛇等；

  6. 虐待动物：如展示摔打或使用其他工具伤害动物的行为，渲染虐待动物的快感。

宣扬不良价值观 #

包含但不限于以下场景：

  1. 挑战公序良俗或游走在社会道德边缘的行为；

  2. 展示违背伦理道德的糜烂生活；

  3. 宣扬不良生活方式和不良流行文化，渲染消极情绪，宣扬颓废的人生观、世界观和价值观；

  4. 宣扬拜金主义和享乐主义；

  5. 宣传和宣扬丧文化、自杀游戏；

  6. 以肯定、赞许的基调或引入模仿的方式表现打架斗殴、羞辱他人、污言秽语；

  7. 恶意对比、引战行为。

诱导低俗 #

渲染低俗及宣扬低级趣味的行为和内容。包含但不限于以下场景：

  1. 对性部位和性行为进行展示、过度描述或内涵表述的；

  2. 有明显的性挑逗、性骚扰、性侮辱或类似效果的画面、台词、音乐及音效等；

  3. 以成人电影、情色电影、三级片、偷拍、走光、露点及各种挑逗性文字或图片作为内容标题、分类或宣传推广；

  4. 发布和传播其他有伤社会风化的内容。

不文明用语 #

包含但不限于以下场景：

  1. 骚扰辱骂：使用污言秽语、蔑称、外号、同音近音词等直接辱骂或言语骚扰他人；

  2. 歧视恶搞：恶搞、歧视特定群体，或言论带有明显的性别歧视、种族歧视、地域歧视等；

  3. 情绪宣泄：通过粗俗不堪的口头禅、网络用语等发泄情绪，表达不满的。

恶意营销 #

以营利、获益为目的，内容附带联系方式、售卖链接，或诱导私信联系转线下交易，引导用户消费但可能存在风险的行为。包含但不限于以下场景：

  1. 联系方式：如电话号码、微信号、公众号、二维码、微信群等；

  2. 营销链接：带有明显营销意图的第三方网址，或诱导用户在第三方电商平台领取红包、购买交易的口令和店铺号等；

  3. 引导交易：

  4. 票务售卖：明确公示价格，买卖转让演唱会门票等；

  5. 虚拟交易：推广虚拟货币、虚拟商品、虚拟服务类交易，如游戏点卡、游戏在线服务、网游装备、QQ号码、Q币、软件序列号、电子书等；

  6. 求资源/诱导私信联系：诱导读者通过私信渠道联系或获取资源（如联系方式、链接、加微信群等）；

  7. 纯广告：以推广产品、盈利为目的作品，包括产品宣传、商业服务宣传、商业赞助宣传、商业展销宣传、商家优惠酬宾宣传等。

标题低质 #

通过夸张描述、隐瞒信息、扭曲原意等方式增加标题吸引力，引诱用户阅读，但与内文真实情况不符，造成用户阅读后心理落差与不良体验。包含但不限于以下场景：

  1. 标题夸张：

    1. 夸张式标题：将感受、范围、结果、程度等夸张夸大描述，造成耸人听闻的效果；

    2. 悬念式标题：标题滥用转折、隐藏关键性信息，营造悬念、故弄玄虚；

    3. 强迫式标题：标题采用挑衅恐吓、强迫建议等方式，诱导用户阅读；

  2. 标题与内容原意有偏差：

    1. 标题存在歧义 ：标题对要素信息的表达不清晰、不对称或不完整，形成冲突，对用户产生误导；

    2. 标题无中生有 ：标题捏造内容中不存在的人、物、情节、言论等，或用猜测、不确定消息诱导用户点击；

    3. 标题滥用名称、题材不明确：标题未明确所涉及题材，滥用演员名代替剧中人物名，或用新闻式标题描述影视剧情节，引起歧义，误导用户认为是明星花边或社会时政负面；

    4. 封面与标题不符： 封面图与内容主体完全不相关，或封面图结合标题易使读者产生误解；

  3. 标题不规范：标题含有错别字或存在语病、滥用标点符号、语义不通顺等，影响理解与阅读体验。

话题无关 #

发布内容与所带话题的内容不相关，如内容未提及话题主体、内容与话题城市异地、只带话题不写内容等。

封面/配图低质 #

封面或配图内容质量差，与文章主体无关且不能为文章增补信息量。包含但不限于以下场景：

  1. 图片与内容毫无关联，且图片质量差的；

  2. 图文明显不相关，带不良图片蹭流量的，图片类型包含但不限于如下：

    1. 有不良诱导倾向的图片；

    2. 偷拍类图片；

    3. 男女亲密同框，易引发低俗、恶俗等不良想象图片等。

内容低质 #

图文格式混乱或不美观，音画质量差等影响阅读体验。包含但不限于以下场景：

  1. 图文低质的内容，包含但不限于以下场景：

    1. 排版混乱：文章乱码、无段落或无标点；

    2. 语意不明：病句或错别字较多、乱码符号，有碍于读者理解内容；

    3. 逻辑混乱：内容拼凑或重复，前后内容没有衔接，无关内容占比较大；

  2. 音画低质的内容，包括但不限于以下场景：

    1. 视频缩放画面：视频中角标/logo/字幕被剪切，导致显示不全，或画面中人物面部被部分剪切；

发布已过时效内容 #

指发布的资讯、消息等是已经过期的内容，并非新近发生的，因时间不明确、隐藏时间词或时间词过期等引起读者理解偏差、混淆了事件的真实发生时间。包含但不限于以下场景：

  1. 旧闻新发：

    1. 明确时间陈旧：时间明确且已过时，事件已结束，近期无后续、无持续影响和相关讨论的；

    2. 时间不明：文章中提到的事件为很久之前发生的，特意隐去时间，易误导他人是新近发生的事件；

  2. 信息陈旧：指事件持续发酵，已经有新进展、数据有更新、有新的结论，但仍发布陈旧的信息；

  3. 时间词过时：有明确的时间限定词，如「今天」「本月」等，而内容发布时间与限定范围不符，容易引起阅读误解的。
